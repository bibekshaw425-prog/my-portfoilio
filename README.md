## ðŸ› ï¸ Getting Started

Follow these steps to set up the project locally:

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bibekshaw425-prog/my-portfoilio.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd bibek-portfolio-pro-6e3 (1) # Or your cloned directory name
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

### Running the Development Server

Start the local server with:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application in action.
